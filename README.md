### Installation
in terminal npm i then after run your schema. Workbench was used here but command line also works. After running schema then npm run seed. Finally we start the server by using node index.js in the command line. 
### Usage
Once server is started use the local host "http://localhost:3001/api" and add on to the url as needed to test.
